FactionEvent est un plugin Minecraft développé pour les serveurs Spigot. Il permet de gérer des événements et interactions dynamiques entre factions, avec des fonctionnalités de suivi des scores, notifications et intégration serveur.

### Fonctionnalités principales
- Gestion d'événements pour différentes factions.
- Suivi des scores et statistiques en temps réel.
- Notifications pour les joueurs impliqués.
- Commandes personnalisées pour administrateurs et joueurs.
- Configuration flexible via fichiers YAML.
